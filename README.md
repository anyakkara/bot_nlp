Telegram бот с системой контекста, подключенный к локальной LLM через LM Studio

## Настройка своего бота

1. **Создайте бота в Telegram:**
 - Напишите @BotFather в Telegram
 - Команда `/newbot`
 - Выберите имя и username для бота
 - Скопируйте полученный токен

2. **Установите зависимости:**
 - pip install pytelegrambotapi requests python-dotenv

3. **Настройте окружение:**
 - Создайте файл .env в корне проекта
 - Добавьте ваш токен: BOT_TOKEN=ваш_уникальный_токен_от_BotFather

4. **Запустите LM Studio:**
 - Запустите LM Studio
 - Загрузите языковую модель

5. **Убедитесь, что сервер работает на http://localhost:1234**

6. **Запустите бота:**
 - bash
 - python script.py


Команды бота
/start - приветствие и список команд
/model - информация о используемой LLM модели
/clear - очистка истории диалога
